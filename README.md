## 📚 About This Repository

This repository is my dedicated space for learning and practicing **JavaScript**, the backbone of modern web development.  
It includes real hands-on code covering everything from **core JS topics** to **DOM manipulation**, **callbacks/promises**, and **mini-projects** like a calculator and todo app with local storage.

---

## 🔧 Technologies Used

- ✅ JavaScript (ES6+)
- ✅ HTML / DOM
- ✅ CSS (for small UI demos)
- ✅ Browser API: `localStorage`, `fetch`, `setTimeout`, etc.
